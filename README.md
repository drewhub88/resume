## What is this?
An automated LaTeX-to-PDF builder for my resume, using a CI/CD workflow (pipeline) built with GitHub Actions.

## Why?
For convenience. So I can keep my resume up to date and organised without the hassle of manually recompiling and renaming files every time I make a change.

## Tech used
- **LaTeX** — resume source (based on an Overleaf template)
- **GitHub Actions** — CI/CD automation
- **xu-cheng/latex-action** — TeX Live compilation in CI
- **YAML** — workflow configuration
